II. 创建表和数据操作

分别创建三张表学生表S,课程表C，和学习表SC。这三个表的字段、主键和外键定义如下：
学生表： S（sno,sname,sex,age,sdept,logn,pswd)
主键： sno

Column Name Data Type Meaning
sno char(4) 学号
sname char(8) 姓名
sex char(2) 性别
age char(2) 年龄
sdept char(10) 所在系
logn char(20) 登录名
pswd char(20) 登录密码

课程表： C（cno,cname,credit,cdept,tname)
主键： cno

Column Name Data Type Meaning
cno char(4) 课程号
cname char(8) 课程名
credit integer 学分
cdept char(10) 院系
tname char(8) 教师名

结果如下：

学习表： SC（sno,cno,grade)
主键： （sno,cno）
外键： sno,cno

Column Name Data Type Meaning
sno char(4) 学号
cno char(4) 课程号
grade integer 成绩
